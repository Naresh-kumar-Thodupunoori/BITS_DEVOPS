Project Overview

This web application is part of a DevOps assignment that demonstrates essential DevOps practices including version control with Git and containerization with Docker.

Technologies Used

HTML5 Modern web structure
CSS3 Styling and layout
Git Version control
Docker Containerization

Assignment Objectives

This assignment covers:

  • Setting up a local Git repository
  • Demonstrating Git commands (init, status, add, commit, log, branch, switch)
  • Integrating with GitHub
  • Creating a Docker image
  • Pushing to Docker Hub
  • Running containers locally
